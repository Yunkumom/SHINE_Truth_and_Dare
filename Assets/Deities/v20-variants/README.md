# v20 Deity Variants / v20 神祇變體

Nine high-resolution PNG source artworks for Encounter Cards v20. Each deity receives a second pose or action, while the Taiwan shape is hidden in a physical detail rather than placed prominently in the clouds. Runtime WebP copies are version-bound under `Development/Source/Main-App-v20/src/assets/deities/`.

本資料夾保存 v20 的九張高解析 PNG 原始圖。每位神祇新增一種姿勢或動作，台灣輪廓藏在具體器物細節中，而非明顯放在雲層。網站使用的 WebP 副本位於 v20 source。
