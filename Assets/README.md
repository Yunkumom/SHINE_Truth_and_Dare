# Assets / 素材庫

This directory is the home for reusable source resources and their governance. `Catalog/` records licences, provenance, attribution, and cultural-safety limits. Version-bound runtime copies remain inside the relevant app or source tree for reproducibility.

本目錄是可重用原始素材與素材治理的集中位置。`Catalog/` 記錄授權、來源、標示與文化安全限制。為確保可重建性，版本綁定的 runtime 副本仍保留於對應 App 或 source 內。

`Deities/` contains the original nine deity portraits; `Deities/v20-variants/` contains nine additional poses/actions. Optimized WebP runtime copies live in the immutable v20 source and current `Development/Source/Main-App-v21/src/assets/deities/`; v21 adds locator metadata without altering the artwork files.

`Deities/` 保存原始九張神祇圖；`Deities/v20-variants/` 保存另外九張姿勢／動作變體。最佳化 runtime WebP 同時保留於不可變 v20 與目前 v21 source；v21 僅新增定位資料，不改動圖片。
