# Assets / 素材庫

This directory is the home for reusable source resources and their governance. `Catalog/` records licences, provenance, attribution, and cultural-safety limits. Version-bound runtime copies remain inside the relevant app or source tree for reproducibility.

本目錄是可重用原始素材與素材治理的集中位置。`Catalog/` 記錄授權、來源、標示與文化安全限制。為確保可重建性，版本綁定的 runtime 副本仍保留於對應 App 或 source 內。

`Deities/` contains the original nine deity portraits; `Deities/v20-variants/` contains nine additional poses/actions. `Deities/v23-taiwan-safe/` contains the current 18 high-resolution edits, each with a recognizable, differently coloured Taiwan silhouette inside the card-safe crop. Optimized WebP runtime copies live in the immutable release sources and current `Development/Source/Main-App-v23/src/assets/deities/`.

`Deities/` 保存原始九張神祇圖；`Deities/v20-variants/` 保存另外九張姿勢／動作變體。`Deities/v23-taiwan-safe/` 保存目前 18 張高解析修正版，每張都在卡片安全裁切範圍內藏有可辨識、不同配色的台灣輪廓。最佳化 runtime WebP 保留於不可變版本 source 與目前 v23 source。
