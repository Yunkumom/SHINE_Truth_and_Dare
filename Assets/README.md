# Assets / 素材庫

This directory is the home for reusable source resources and their governance. `Catalog/` records licences, provenance, attribution, and cultural-safety limits. Version-bound runtime copies remain inside the relevant app or source tree for reproducibility.

本目錄是可重用原始素材與素材治理的集中位置。`Catalog/` 記錄授權、來源、標示與文化安全限制。為確保可重建性，版本綁定的 runtime 副本仍保留於對應 App 或 source 內。

`Deities/` contains the original nine deity portraits; `Deities/v20-variants/` contains nine additional poses/actions. Optimized WebP runtime copies live in the immutable v20–v21 sources and current `Development/Source/Main-App-v22/src/assets/deities/`; v22 preserves the locator metadata without altering the artwork files.

`Deities/` 保存原始九張神祇圖；`Deities/v20-variants/` 保存另外九張姿勢／動作變體。最佳化 runtime WebP 保留於不可變 v20–v21 與目前 v22 source；v22 延續定位資料，不改動圖片。
