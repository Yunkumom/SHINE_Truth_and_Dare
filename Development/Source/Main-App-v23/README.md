# Main App v23 Source / 主應用程式 v23 原始碼

This is the React, TypeScript, and Vite source for Encounter Cards v23. It adds a privacy-safe direct-manipulation layout editor, a thresholded swipe-and-flip deck, a larger 63:88 card, and selectable contact exchange while preserving the v21 hidden-Taiwan interaction.

此處保存 Encounter Cards v23 source；加入不含個資的拖曳版面編輯器、滑動抽卡與翻面、放大的 63:88 卡片及可選擇的聯絡資訊交換，同時保留 v21 藏台灣互動與 430 × 932 置中適配。

Restore dependencies with `npm ci`, then use `npm run typecheck`, `npm run lint`, `npm test`, and `npm run build`. Verified v23 outputs become immutable; later behavior must use the next numbered version.
