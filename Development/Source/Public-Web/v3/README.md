# Public Web v3 source recipe

Public Web v3 is produced from the canonical `Development/Source/Main-App-v20/` source. Run its verified build, then run `Development/Automation/Scripts/finalize-public-v3.mjs` exactly once. The generated output at `Apps/Public-Web/v3/` becomes immutable after verification and publication.

公開版 v3 由正式的 `Main-App-v20` 原始碼產生。完成驗證後執行 v3 finalizer；驗證與發布後 `Apps/Public-Web/v3/` 不得覆寫。
