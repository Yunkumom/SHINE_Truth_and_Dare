# Public Web v4 source recipe

Public Web v4 is produced from the canonical `Development/Source/Main-App-v21/` source. Run its verified build, then run `Development/Automation/Scripts/finalize-public-v4.mjs` exactly once. The generated output at `Apps/Public-Web/v4/` becomes immutable after verification and publication.

公開版 v4 由正式的 `Main-App-v21` 原始碼產生。完成驗證後執行 v4 finalizer；驗證與發布後 `Apps/Public-Web/v4/` 不得覆寫。
