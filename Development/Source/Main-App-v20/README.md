# Main App v20 Source / 主應用程式 v20 原始碼

This is the React, TypeScript, and Vite source for Encounter Cards v20. It restores the v16 entrance and mythic collectible-card presentation, introduces an 18-image deity library, independently composes questions/artwork/blessings, and preserves centered 430 × 932 desktop fitting.

此處保存 Encounter Cards v20 source；重建 v16 入口與神話收藏卡質感，加入 18 張神祇圖庫、問題／圖像／祝福獨立組合，以及置中的 430 × 932 桌面適配。

Restore dependencies with `npm ci`, then use `npm run typecheck`, `npm run lint`, `npm test`, and `npm run build`. Verified v20 outputs become immutable; later behavior must use the next numbered version.
