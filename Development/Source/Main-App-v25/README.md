# Main App v25 Source / 主應用程式 v25 原始碼

This is the React, TypeScript, and Vite source for Encounter Cards v25. It preserves the privacy-safe layout editor, swipe-and-flip deck, 63:88 card, and selectable contact exchange while replacing the old locator symbol with a transparent, Natural Earth-derived dual-stroke Taiwan coastline glow.

此處保存 Encounter Cards v25 source；保留不含個資的拖曳版面編輯器、滑動抽卡與翻面、63:88 卡片及可選擇的聯絡資訊交換，並將舊定位符號替換成透明、源自 Natural Earth 的雙線台灣海岸線發光，同時維持 430 × 932 置中適配。

Restore dependencies with `npm ci`, then use `npm run typecheck`, `npm run lint`, `npm test`, and `npm run build`. Verified v25 outputs become immutable; later behavior must use the next numbered version.
