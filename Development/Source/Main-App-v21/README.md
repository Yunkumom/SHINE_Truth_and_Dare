# Main App v21 Source / 主應用程式 v21 原始碼

This is the React, TypeScript, and Vite source for Encounter Cards v21. It preserves the v16-inspired presentation and adds per-artwork hidden-Taiwan hotspots with an accessible 600 ms long-press reveal and 3-second blinking locator.

此處保存 Encounter Cards v21 source；延續 v16 入口與神話收藏卡質感，為 18 張神祇圖加入各自的藏台灣定位、600ms 長按揭示與 3 秒閃爍標記，並維持置中的 430 × 932 桌面適配。

Restore dependencies with `npm ci`, then use `npm run typecheck`, `npm run lint`, `npm test`, and `npm run build`. Verified v21 outputs become immutable; later behavior must use the next numbered version.
