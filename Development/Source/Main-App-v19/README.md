# Main App v19 Source / 主應用程式 v19 原始碼

This is the React, TypeScript, and Vite source for Encounter Cards v19. It preserves the 430 × 932 desktop-fit contract and adds the Taiwan deity artwork/card export presentation.

此處保存 Encounter Cards v19 的 React／TypeScript／Vite source；保留 430 × 932 桌面適配，並加入台灣神祇圖像與卡片輸出。

Restore dependencies with `npm ci`, then use `npm run typecheck`, `npm run lint`, and `npm test`. v19 generated outputs are immutable after verification; later behavior must use v20.

先用 `npm ci` 還原依賴，再執行 typecheck、lint 與 test。v19 驗證後不可覆寫；後續行為需建立 v20。
