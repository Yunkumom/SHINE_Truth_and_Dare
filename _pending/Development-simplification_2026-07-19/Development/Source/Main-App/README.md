# Main App Source / 主應用程式原始碼

Maintain Encounter Cards v17 here with React, TypeScript, and Vite. `src/` is authored code, `public/` contains approved PWA resources, and `dist/` is generated. Run `npm run typecheck`, `npm run lint`, `npm test`, and `npm run build:standalone` from this directory.

此處維護 Encounter Cards v17。`src/` 是人工程式碼，`public/` 保存核准 PWA 資源，`dist/` 是生成內容；不可手改生成成品。
